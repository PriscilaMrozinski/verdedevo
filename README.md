# 🌿 Verde de Vó

Sabe aquelas plantinhas no quintal da vó…  
o chá pra dor de barriga, o cheirinho de hortelã, o alecrim dourado dando aquele sabor especial no peixe…  
a comida feita com carinho, cheia de significado, e o cuidado simples que sempre funcionava?

O **Verde de Vó** nasce dessa memória afetiva da conexão entre natureza, cuidado e conhecimento popular que atravessa gerações.

Mais do que um sistema, ele é uma forma de resgatar esse saber e torná-lo acessível no mundo digital.

Esse projeto faz parte da história e do cotidiano de muitas pessoas. Com a ideia de unir esse “dia a dia” saboroso e saudável, treino, pratico e exploro códigos que transformam e facilitam o acesso à informação, ajudando na escolha e no uso consciente das plantas, com visão de evolução para uma plataforma digital completa e acessível ao público.

No fim, este projeto é exatamente isso: um encontro entre tradição, aprendizado e inovação.


---

## 💻 Sobre o projeto

Sistema web para **consulta e gestão de plantas medicinais**, com:

- painel administrativo (CRUD)  
- busca inteligente por termos, sintomas e linguagem natural  
- mecanismo de relevância com técnicas de PLN (normalização, similaridade e radicais)  
- dashboard interativo com visualização de dados  

---

## 🚀 Tecnologias utilizadas

- Python (Flask)  
- JavaScript  
- HTML  
- Tailwind CSS  
- Chart.js  
- Banco de dados em JSON  
- NLTK (Processamento de Linguagem Natural)  

---

## 🤖 Inteligência aplicada (PLN)

O projeto já aplica conceitos de **Processamento de Linguagem Natural (PLN)**, como:

- normalização de texto (remoção de acentos e variações)  
- comparação por similaridade (correção de erros de digitação)  
- uso de radicais com **RSLPStemmer (NLTK)**  
- mapeamento de sinônimos de sintomas  
- busca baseada em intenção do usuário  

---

## 🌱 Objetivo

Unir tecnologia e saber popular para facilitar o acesso a informações sobre plantas medicinais, valorizando práticas tradicionais através de soluções digitais.
